# 📘 Day 02 — Linear Regression

## 📖 Overview

Linear Regression is one of the most fundamental and widely used supervised learning algorithms. It models the relationship between a dependent variable and one or more independent variables.

---

## 🧠 Concept

Linear Regression attempts to fit a straight line that best predicts the target variable.

Where:

* y = Dependent variable (target)
* x = Independent variable (feature)
* m = Slope of the line
* b = Intercept

---

## 🎯 Objective

Minimize the error between actual and predicted values using a cost function.

MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2

---

## ⚙️ How It Works

1. Fit a line to training data
2. Calculate prediction error
3. Adjust parameters (m, b)
4. Minimize error using optimization

---

## 🌍 Applications

* House price prediction
* Sales forecasting
* Trend analysis

---

## 📊 Key Takeaways

* Simple yet powerful algorithm
* Works best with linear relationships
* Sensitive to outliers
