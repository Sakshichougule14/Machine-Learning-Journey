## 🧠 What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence that enables systems to learn patterns from historical data and make predictions or decisions without explicit programming.

Formally, a program is said to learn from experience **E** with respect to some task **T** and performance measure **P**, if its performance on **T**, as measured by **P**, improves with experience **E**.

---

## 🔍 Types of Machine Learning

### 1. Supervised Learning

* Uses labeled datasets
* Objective: Learn mapping function from input to output
* Common algorithms: Linear Regression, Logistic Regression

### 2. Unsupervised Learning

* Uses unlabeled data
* Objective: Identify hidden patterns or structures
* Common algorithms: K-Means Clustering, Hierarchical Clustering

### 3. Reinforcement Learning

* Agent interacts with environment
* Learns via reward and penalty mechanism
* Applications: Robotics, Game AI
